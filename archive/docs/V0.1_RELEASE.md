## ğŸ¯ Usage

### For Streamers
1. Go to StreamElements > My Overlays
2. Add Custom Widget
3. Copy contents from `widget/` folder
4. Enter your Bungie name (e.g., Marty#2689)
5. Customize appearance
6. Add to your stream scene

### For Developers
```bash
# Clone repository
git clone https://github.com/martinricard/d2loadout-widget

# Install backend dependencies
cd backend
npm install

# Set up environment
cp .env.example .env
# Add your Bungie API key

# Run locally
npm start

# Deploy to Render
git push origin main
```
