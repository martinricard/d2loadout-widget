Setup
üîç Finding Your Bungie ID: Go to bungie.net ‚Üí Sign in ‚Üí Click your profile ‚Üí Your Bungie Name is shown as Username#1234
  1. Enter your Bungie ID (e.g., Guardian#1234) in the widget settings
  2. Choose display mode: "Always Visible" (widget stays on screen) or "Auto-Hide" (widget appears only when triggered)
  3. If using Auto-Hide, set Display Duration (default: 15 seconds) and Command Cooldown (default: 20 seconds)
  4. Select trigger type: Chat Commands or Channel Point Redeems
Chat Commands
  • !loadout - Show full loadout
  • !weapons - Show weapons only
  • !armor - Show armor only
  • !stats - Show stats only
  • !subclass - Show subclass only
  • !artifact - Show artifact only
  • !dimlink - Post DIM loadout link in chat
Channel Points
About Channel Points: Create channel point rewards with names like "Show Loadout", "Show Weapons", "Show Stats", etc. The widget will automatically detect and display the corresponding view.
Cooldown System
‚è±Ô∏è Command Cooldown: Prevents spam by ignoring repeated commands during the cooldown period (default: 20 seconds). Set higher than Display Duration (15s) to avoid triggering the widget before it hides. Adjustable from 0-60 seconds in settings. Applies to both chat commands and channel points.
DIM Link Feature
üîó About the DIM Link: The widget displays a shortened link to your loadout in DIM (Destiny Item Manager). Use the !dimlink chat command to post the link in chat. This requires setting up a StreamElements custom command (bot posts to chat, not the widget). The link uses TinyURL and is automatically updated with your current loadout.
üí° Pro Tip: Customize command names, display duration, and cooldown in the widget settings. The widget automatically refreshes every minute to show your latest Destiny 2 loadout!

Privacy Policy

Last Update: October 11, 2025
D2 Loadout Widget is a free StreamElements widget and chat bot integration for Destiny 2 streamers. We explain how your data is used below.
  • Bungie API We use the Bungie.net API to fetch your Destiny 2 loadout information. Your Bungie ID is passed through our server to make API requests, but is not stored or logged. No personal data is retained.
  • TinyURL Service We generate shortened DIM (Destiny Item Manager) links for easy sharing in chat and overlays. TinyURL may log generated URLs for their service operations.
  • Chat Commands Our bot responds to chat commands (e.g., !dimlink) to share your loadout link. The command uses your configured Bungie ID to fetch real-time data. No chat messages are stored.
  • StreamElements This widget runs on StreamElements. Please refer to their privacy policy for information on data collection by the platform.
  • Server Hosting Our backend is hosted on Render.com. Temporary logs may be created for debugging but contain no personal information and are automatically deleted after 7 days.
